Meter

O elemento <meter> é utilizado para representar um valor escalar dentro de um intervalo conhecido. É ideal para mostrar medições que têm um valor máximo e um mínimo definidos, como:

  • Medidas de desempenho: Por exemplo, uso de CPU ou memória em um aplicativo
  • Avaliações: Como a porcentagem de conclusão de uma tarefa ou projeto em relação a um total.
  • Resultados de medições: Como a quantidade de chuva em um determinado período em comparação com um valor máximo.

Características

  • Valor Escalar: O <meter> é mais apropriado quando você quer indicar um valor que faz parte de um conjunto de dados, como uma porcentagem ou uma medição específica.
  • Interatividade: Pode incluir etiquetas e descrições que ajudam a contextualizar o valor apresentado.
            
Exemplo de código

                
                    <meter max="100" min="0" value="25"<>/meter>

Progress

O elemento <progress> é usado para mostrar o progresso de uma tarefa em andamento. É ideal para:

  • Indicadores de carregamento: Mostrando o progresso de download ou upload de arquivos.
  • Operações de longa duração: Como o processamento de uma tarefa ou a instalação de software.

Características

  • Progressão: O <progress> é mais indicado quando você deseja mostrar um estado de progresso em tempo real, que pode ser atualizado à medida que a tarefa avança.
  • Simples e Direto: É geralmente usado para indicar uma operação em andamento, e o valor final é muitas vezes conhecido, como 100% para concluir.
            
Exemplo de código

                
                    <progress id="progress" max="100" min="0" value="0"></progress>
            
            <script>
                var progress = document.getElementById('progress');
                const interval = setInterval(() => {
                    progress.value += 10;
                    if (progress.value >= 100) {
                        alert('concluído')
                        clearInterval(interval)
                    }
                        
                }, 1000);
        </script>
