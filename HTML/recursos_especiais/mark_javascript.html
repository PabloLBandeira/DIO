Mark usando Javascript

A tag <mark> é utilizada em HTML para destacar partes do texto que são importantes ou relevantes. Ela serve como uma forma de chamar a atenção do leitor para informações específicas.

  • Destaque Visual:O texto envolvido pela tag <mark> geralmente aparece com um fundo amarelo, facilitando a identificação de conteúdo importante.
  • Uso Semântico:Usar <mark> indica que o texto possui um significado especial, como termos-chave em um artigo ou resultados de busca.
  • Acessibilidade:A tag <mark> melhora a acessibilidade, permitindo que tecnologias assistivas, como leitores de tela, identifiquem rapidamente o texto destacado.

            <script>
                const input = document.getElementById('busca');
                const texto = document.getElementById('texto');
        
                input.addEventListener('input', () => {
                    const termoBusca = input.value.toLowerCase();
                    const regex = new RegExp(`\\b${termoBusca}\\b`, 'gi');
                    texto.innerHTML = texto.textContent.replace(regex, '<mark>$&</mark>');
                });
            </script>

O desenvolvimento de software é um processo complexo e dinâmico que envolve a criação, manutenção e aprimoramento de sistemas e aplicativos. Este campo abrange várias etapas, desde a concepção da ideia até a implementação e testes. A fase inicial, chamada de levantamento de requisitos, é crucial, pois permite entender as necessidades dos usuários e as funcionalidades desejadas.Uma vez definidos os requisitos, inicia-se o design do software, onde arquitetos e desenvolvedores planejam a estrutura e a interação dos componentes. O desenvolvimento propriamente dito envolve a codificação, onde os programadores transformam as especificações em linhas de código, utilizando diferentes linguagens de programação como Python, Java e JavaScript.
