Mark usando Javascript

A tag <mark> é utilizada em HTML para destacar partes do texto que são importantes ou relevantes. Ela serve como uma forma de chamar a atenção do leitor para informações específicas.

  • Destaque Visual:O texto envolvido pela tag <mark> geralmente aparece com um fundo amarelo, facilitando a identificação de conteúdo importante.
  • Uso Semântico:Usar <mark> indica que o texto possui um significado especial, como termos-chave em um artigo ou resultados de busca.
  • Acessibilidade:A tag <mark> melhora a acessibilidade, permitindo que tecnologias assistivas, como leitores de tela, identifiquem rapidamente o texto destacado.
