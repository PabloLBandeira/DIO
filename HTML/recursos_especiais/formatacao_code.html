code {

Formatação semântica

Texto formatado de maneira semântica.

Para copiar esta informação, pressione CTRL + C.

Para colar em seu arquivo pressione CTRL + V

Para estilizar seu HTML use:

                    
// copie e cole no seu arquivo .css

                        
                            code {
                                font-style: italic;
                                background-color: lightgrey;
                                color: red;
                            }

Details e summary

Usada para ganhar espaço de tela expandindo blocos de texto sem a necessidade de usar Javascript

Como estilizar o HTML? 
                    
                        kbd {
                            border: solid 1px black;
                        }
                        
                        code {
                            font-style: italic;
                            background-color: lightgray;
                            color: red;
                        }
                        
                        pre {
                            background-color: lightgray;
                        }
                        
                        pre code {
                            font-style: italic;
                            background-color: lightgrey;
                            color: red;
                        }
                        
                        pre p {
                            font-style: italic;
                            color: lightcoral;
                        }
