Code, kbd e pre

Esses elementos ajudam a organizar e apresentar informações técnicas de maneira clara e acessível, melhorando a experiência do usuário em documentos e páginas web.

code

O elemento <code> é usado para representar trechos de código de programação ou texto que deve ser exibido em uma fonte monoespaçada. É ideal para:

  • Código em Documentação: Quando você deseja destacar um pequeno trecho de código em um contexto maior, como em tutoriais, artigos ou documentação.
  • Exibição de Sintaxe: Para mostrar expressões específicas, comandos ou funções de uma linguagem de programação.

Formatação não semântica

Texto formatado sem levar em consideração a otimização semântica.

                        
Exemplo de código

                        <blockquote>
                            <p>Para estilizar seu HTML use:</p>
                            <p>code {<br>
                                font-style: italic;<br>
                                background-color: lightgrey;<br>
                                color: red;<br>
                            }</p>
                        </blockquote>

Visão do usuário

Para estilizar seu HTML use:

code {
font-style: italic;
background-color: lightgrey;
color: red;
}

Formatação semântica

Texto formatado de maneira semântica.

                        
Exemplo de Código

                        
                            <code>
                                code {
                                    font-style: italic;
                                    background-color: lightgrey;
                                    color: red;
                                    } 
                            </code>
                        
                    
                            
Visão do usuário

                                
                                    code {
                                        font-style: italic;
                                        background-color: lightgrey;
                                        color: red;
                                        }

kbd

O elemento <kbd> é utilizado para representar a entrada do teclado. É ideal para:

  • Comandos e Atalhos de Teclado: Quando você quer indicar quais teclas devem ser pressionadas pelo usuário, como combinações de teclas ou sequências.
  • Instruções Interativas: Para destacar interações que dependem de entradas do teclado.

Formatação não semântica

                            
Exemplo de código

                            Para copiar esta informação, pressione <b><i>CTRL</i></b> + <b><i>C</i></b>.
                        
                            
Visão do usuário

                            Para copiar esta informação, pressione CTRL + C

Formatação semântica

                            
Exemplo de Código

                            Para copiar esta informação, pressione <kbd> CTRL</kbd> + <kbd>C</kbd>.
                        
                            
Visão do usuário

                            Para copiar esta informação, pressione  CTRL + C.

pre

Use a tag <pre> quando precisar que a formatação original do texto seja mantida, como em códigos de programação ou na apresentação de dados estruturados.

  • Preservação de Espaços: O texto dentro da tag <pre> não ignora espaços em branco adicionais, ao contrário do que ocorre em outras tags HTML.
  • Quebras de Linha: Quebras de linha são respeitadas, permitindo que o texto apareça em várias linhas conforme definido.
  • Fonte Monoespaçada: O texto dentro de <pre> geralmente é exibido em uma fonte monoespaçada (como Courier), o que facilita a leitura de código ou de textos que exigem alinhamento.
                    
Exemplo de Código

                    
                        <pre>
                            <code>
                            <li><b>Comandos e Atalhos de Teclado:</b>
                            </code>
                        </pre>
                    
                
                    
Visão do usuário

                    
                        
  • Comandos e Atalhos de Teclado:
