Tag audio


O que é?

A tag <audio> é um elemento HTML5 usado para incorporar áudio em uma página web. Ela fornece uma maneira simples e eficiente de adicionar arquivos de áudio, como música, podcasts, ou efeitos sonoros, sem a necessidade de plugins externos.

Quando usar?

A tag <audio> é projetada para ser usada de forma semântica, oferecendo uma interface padrão para reprodução de áudio e controle.
