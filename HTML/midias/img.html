Gif

Gif são várias imagens interpoladas que, ao serem reproduziadas em sequência, assemelham-se a um vídeio

Jpeg

JPEG (Joint Photographic Experts Group) é um formato de compressão de imagens. ideal para fotos e imagens com muitos detalhes e cores. Ele reduz o tamanho do arquivo ao comprimir a imagem, mas com perda de qualidade. Umas das suas características marcantes é preencher o fundo com uma cor sólida, o que fica bem explícito no exemplo abaico

Fundo alterado para ser possível visualizar o funcionamento do .jpeg



PNG

PNG (Portable Network Graphics) é um formato de imagem sem perdas, ou seja, comprime arquivos sem perder qualidade. Ele suporta transparência, o que o torna ideal para gráficos e logotipos na web.

Esta péssima escolha de cor, mostra perfeitamente como o .png utiliza a transparência


SVG

SVG (Scalable Vector Graphics) é um formato de imagem vetorial baseado em XML. Ele permite redimensionamento sem perder qualidade e é ideal para gráficos, ícones e logos na web. Além disso, pode ser manipulado via CSS e JavaScript, tornando-o flexível e interativo.
