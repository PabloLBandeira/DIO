O que é

HTML (HyperText Markup Language) é a linguagem padrão usada para criar páginas da web. Ele organiza o conteúdo em elementos como títulos, parágrafos, links e imagens, definindo a estrutura e a semântica da página. Navegadores interpretam o HTML para exibir as páginas aos usuários.

Inspetor de elementos

O Inspetor de Elementos é uma ferramenta dos navegadores que permite ver e editar o código HTML, CSS e JavaScript de um site em tempo real. Com ele, você pode entender como as páginas são construídas, testar alterações visuais ou funcionais, e até corrigir erros no código sem precisar modificar o arquivo original. É super útil para aprender como tudo funciona na prática e experimentar mudanças antes de aplicá-las no projeto real.

Estrutura de elementos

A estrutura de elementos em HTML é como o esqueleto de uma página web. Cada elemento é uma "caixinha" que representa algo, como um título, um parágrafo, uma imagem ou um link. Os elementos começam com uma tag de abertura (tipo <p> para parágrafos), têm o conteúdo dentro, e depois fecham com uma tag de fechamento (como <p>). Esses blocos organizam o conteúdo para que o navegador saiba o que mostrar e como mostrar.

Tags

As tags são como etiquetas que indicam o que cada parte do conteúdo vai ser na página. Elas sempre ficam entre os sinais de "menor que" (<) e "maior que" (>). Por exemplo, a tag h1 indica um título principal, enquanto p indica um parágrafo. Quase todas as tags têm uma versão de abertura (<p>) e uma de fechamento (<p>), e o conteúdo vai no meio delas. Cada tag tem um propósito, como estruturar texto, criar links ou inserir imagens.

Textos

O texto dentro do HTML é o conteúdo que vai ser exibido na página para as pessoas. Esse texto é "organizado" pelas tags, que dizem ao navegador como mostrar cada parte. Por exemplo, se você colocar o texto dentro da tag `<<h1>>`, ele vai aparecer como um título grande, e se estiver dentro de "<p>", vai aparecer como um parágrafo normal. Além disso, temos tags como "<strong>", que deixa o texto em negrito para destacar algo importante; "<i>", que deixa o texto em itálico; e <small>, que faz o texto ficar menor, geralmente usado para notas ou informações menos importantes. Essas tags ajudam a formatar o texto de formas específicas.

Listas ordenadas e não ordenadas

As listas em HTML, como o próprio nome sugere são usadas para organizar itens em um formato de lista. Existem dois tipos principais:

Lista ordenada: É uma lista em que os itens aparecem em uma ordem específica e são numerados automaticamente. Cada item é colocado dentro de uma tag <li> (list item), e o navegador exibe os números automaticamente.

Exemplo:

entrada

            <ol>
              <li>item 1</li>
              <li>item 2</li>
              <li>item 3</li>
            </ol>
saída
  1. item 1
  2. item 2
  3. item 3

Lista mão ordenada: É uma lista em que os itens não têm uma ordem específica e são exibidos com marcadores.

entrada

            <ul>
              <li>item 1</li>
              <li>item 2</li>
              <li>item 3</li>
            </ul>
saída
  • item 1
  • item 2
  • item 3

Links

Os links em HTML são elementos que permitem a navegação entre páginas, documentos ou recursos na web. Eles são criados com a tag <a>, que significa "anchor" (âncora), e são usados para conectar o conteúdo de uma página a outra página ou arquivo.

Estrutura básica do link:

  • <a>: tag que define o link.
  • <href>: endereço de destino do link, pode ser um site externo, uma página dentro do próprio site, uma sessão da página ou ainda links para download.
  • <texto>: texto visível ao usuário, é a parte clicável.

Tipos de link:

Externo: Direciona o usuário para outro site.

Entrada:
<a href="https://www.google.com">Visite o Google</a>
Saída:
Visite o Google

Interno: Direciona o usuário para outra página dentro do próprio site.

Entrada
<a href="home.html"> Home </a>
Saída
Home

Âncora: Direciona o usuário para uma sessão específica da própria página.

Entrada

link que leva à sessão: <a href="#inicio">Início</a>

sessão apontada pelo link: id="inicio"inicio

Saída

link que leva à sessão: Início

Atríbutos importantes

Existem alguns atributos que podem ser usado nos links, eles são usados para guiar o tipo de navegação que o usuário terá. Abaixo alguns exemplos:

target="_blank": faz com que o link seja aberto em uma nova aba ou janela do navegador.

rel="noopener noreferrer": usado junto com target="_blank" para melhorar a segurança ao abrir links externos, evitando que a página de destino tenha acesso à página de origem.
